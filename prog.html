Introduction to Pseudocode / 伪代码介绍

code What is Pseudocode? / 什么是伪代码？
Pseudocode is a method of writing up a set of instructions for a computer program using plain English.
伪代码是一种使用简单英语编写计算机程序指令集的方法。

Variables and Constants / 变量和常量

storage Variables / 变量
A named memory location that holds a value which can change during program execution.
一个命名的内存位置，用于存储在程序执行期间可以更改的值。
check_circle
Must be declared before use
check_circle
Can be assigned new values
check_circle
Follow naming conventions
DECLARE score : INTEGER
score ← 0
score ← score + 10
lock Constants / 常量
A named memory location that holds a value which cannot change during program execution.
一个命名的内存位置，用于存储在程序执行期间不能更改的值。
check_circle
Declared once, used many times
check_circle
Usually written in uppercase

CONST TAX_RATE ← 0.15

calculate Common Arithmetic Operators / 常见算术运算符
Arithmetic operators allow numerical operations to be performed on values in programming.
算术运算符允许在编程中对值执行数值运算。
Operation / 运算 Operator / 运算符 Example / 示例 Result / 结果
Addition
加法
 + result ← 5 + 3 result = 8
Subtraction
减法
 - result ← 10 - 4 result = 6
Multiplication
乘法
 * result ← 7 * 3 result = 21
Real Division
实数除法
 / result ← 10 / 4 result = 2.5
Integer Division
整数除法
 DIV result ← 10 DIV 4 result = 2
Remainder

 MOD result ← 10 MOD 4 result = 2

assignment Complete this homework assignment to reinforce your understanding of programming concepts.

edit Homework Questions / 作业问题
1. Variables and Constants
变量和常量

- Declare a variable named studentName of type STRING
- Declare a constant named MAX_SCORE with value 100
- Assign a value to the variable
- Display the variable and constant values

数据类型

- A person's age
- A product price
- A student's grade (A, B, C, etc.)
- Whether a student has passed (TRUE/FALSE)
- A single character for a menu option
3. Input/Output
输入/输出

- Ask for a user's name
- Ask for a user's age

4. Arithmetic Operators
算术运算符

- 17 DIV 5
- 17 MOD 5
- Write pseudocode to convert 125 minutes to hours and minutes

顺序结构

- Calculate sum

grading Mark Scheme / 评分标准
1.
Correctly declare variables and constants
2 marks for correct syntax, 1 mark for each correct declaration
2.
Identify appropriate data types
1 mark per correct data type (max 5 marks)
3.
Demonstrate input/output operations
2 marks for correct input, 2 marks for correct output
4.
Use arithmetic operators correctly
1 mark per correct calculation (max 4 marks)
5.
Arrange instructions in correct sequence
1 mark per correctly sequenced step (max 4 marks)
Total: 20 marks / 总分：20分
