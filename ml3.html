assignment Exercises

groups Classification vs. Clustering Activity

In this activity, students will work in groups to classify and cluster different objects, demonstrating their understanding of these two machine learning techniques.

1

Form Groups

Divide students into groups of 3-4

2

Classification Task

Provide each group with a set of objects (e.g., fruits, shapes, or images) and predefined categories. Ask them to classify each object into the correct category.

3

Clustering Task

Give each group a different set of objects without predefined categories. Ask them to create their own groups based on similarities they observe.

4

Discuss Differences

Have each group present their classifications and clusters, discussing the differences between the two approaches.
