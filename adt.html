search Linear Search / 线性搜索

search Definition and Steps / 定义和步骤

A linear search starts with the first value in a dataset and checks every value one at a time until all values have been checked. It can be performed even if the values are not in order.
线性搜索从数据集中的第一个值开始，一次检查每个值，直到检查完所有值。即使值不是有序的，也可以执行。
search

Order Independent / 顺序无关
Works on both sorted and unsorted data
适用于有序和无序数据
1
Check First Value / 检查第一个值
Start with the first element in the array

2
Compare / 比较
If it matches the target, stop and return the position

3
Move to Next / 移至下一个
If not a match, move to the next element
如果不匹配，移至下一个元素
4
Repeat / 重复
Continue until all elements are checked or target is found
继续直到检查完所有元素或找到目标

sort Bubble Sort / 冒泡排序

swap_vert Definition and Steps / 定义和步骤

A bubble sort is a simple sorting algorithm that starts at the beginning of a dataset and checks values in 'pairs' and swaps them if they are not in the correct order. One full run of comparisons from beginning to end is called a 'pass'.
冒泡排序是一种简单的排序算法，从数据集的开始处检查"成对"的值，如果它们的顺序不正确就交换它们。从开始到结束的完整比较过程称为"一次遍历"。
swap_vert
Pairwise Comparison / 成对比较
Compares adjacent elements and swaps if needed
比较相邻元素，需要时进行交换
repeat
Multiple Passes / 多次遍历
May require several passes to fully sort
可能需要多次遍历才能完全排序
speed
Early Termination / 提前终止
Stops when no swaps are made in a pass
当一次遍历中没有交换时停止
1
Compare First Pair / 比较第一对
Check the first two values in the dataset
检查数据集中的前两个值
2
Swap if Needed / 需要时交换
If they're in wrong order, swap them
如果顺序错误，交换它们
3
Move to Next Pair / 移至下一对
Compare the next adjacent pair
比较下一对相邻元素
4
Complete Pass / 完成遍历
Continue until end of dataset
继续直到数据集末尾
5
Repeat or Stop / 重复或停止
If swaps made, start new pass; otherwise stop
如果进行了交换，开始新的遍历；否则停止

code Example Pseudocode / 伪代码示例

DECLARE Numbers : ARRAY[0:5] OF INTEGER // 数组声明
DECLARE Temp : INTEGER // 临时变量
DECLARE Pass : INTEGER // 遍历计数器
DECLARE Index : INTEGER // 索引
DECLARE Swapped : BOOLEAN // 是否交换

// Initialise the array / 初始化数组
Numbers[0] ← 5
Numbers[1] ← 2
Numbers[2] ← 4
Numbers[3] ← 1
Numbers[4] ← 6
Numbers[5] ← 3

// Bubble sort algorithm / 冒泡排序算法
FOR Pass ← 1 TO 5
  Swapped ← FALSE
  FOR Index ← 0 TO 4
    IF Numbers[Index] > Numbers[Index + 1] THEN
      Temp ← Numbers[Index]
      Numbers[Index] ← Numbers[Index + 1]
      Numbers[Index + 1] ← Temp
      Swapped ← TRUE
    ENDIF
  NEXT Index
  IF Swapped = FALSE THEN
    // List is already sorted / 列表已排序
    EXIT
  ENDIF
NEXT Pass

// Output the sorted array / 输出排序后的数组
FOR Index ← 0 TO 5
  OUTPUT Numbers[Index]
NEXT Index
Identifier / 标识符 Data Type / 数据类型 Description / 描述
Numbers ARRAY[0:5] OF INTEGER Stores the list of numbers to sort / 存储要排序的数字列表
Temp INTEGER Temporary variable for swapping / 用于交换的临时变量
Pass INTEGER Tracks number of passes through array / 跟踪数组遍历次数
Index INTEGER Current position being compared / 当前比较的位置
Swapped BOOLEAN Indicates if a swap occurred / 指示是否发生交换
