String and Character Functions

arrow_back
LEFT
Returns leftmost x characters from string
LEFT("ABCDEFGH", 3) returns "ABC"
arrow_forward
RIGHT
Returns rightmost x characters from string
RIGHT("ABCDEFGH", 3) returns "FGH"
content_cut
MID
Returns substring of length y starting at position x
MID("ABCDEFGH", 2, 3) returns "BCD"
straighten
LENGTH
Returns integer value representing string length
LENGTH("Happy Days") returns 10
text_lowercase
LCASE
Returns lower-case equivalent of character
LCASE('W') returns 'w'
text_uppercase
UCASE
Returns upper-case equivalent of character
UCASE('a') returns 'A'
format_size
TO_UPPER
Returns string with all characters in upper case
TO_UPPER("Error 803") returns "ERROR 803"
format_size
TO_LOWER
Returns string with all characters in lower case
TO_LOWER("JIM 803") returns "jim 803"
123
NUM_TO_STR
Returns string representation of numeric value
NUM_TO_STR(87.5) returns "87.5"
text_fields
STR_TO_NUM
Returns numeric representation of string
STR_TO_NUM("23.45") returns 23.45
check_circle
IS_NUM
Returns TRUE if string represents valid numeric value
IS_NUM("-12.36") returns TRUE
code
ASC
Returns ASCII value of character
ASC('A') returns 65
text_format
CHR
Returns character whose ASCII value is x
CHR(65) returns 'A'

Numeric Functions

exposure_zero
INT
Returns the integer part of a real number by truncating the decimal portion
INT(27.5415) returns 27
casino
RAND
Returns a random real number in the range 0 to x (not inclusive of x)
RAND(87) could return 35.43

Date Functions

calendar_today
DAY
Returns the current day number from a date
DAY(04/10/2003) returns 4
date_range
MONTH
Returns the current month number from a date
MONTH(04/10/2003) returns 10
event
YEAR
Returns the current year number from a date
YEAR(04/10/2003) returns 2003
view_week
DAYINDEX
Returns the day index number where Sunday = 1, Monday = 2, etc.
DAYINDEX(12/05/2020) returns 3
Tuesday
edit_calendar
SETDATE
Returns a date variable with the value of Day/Month/Year
SETDATE(26, 10, 2003) returns 26/10/2003
today
TODAY
Returns a date variable corresponding to the current date
TODAY() returns current date

Text File Functions

description
EOF
Returns TRUE if there are no more lines to be read from file FileName
EOF("data.txt") returns TRUE when at end of file
Note: The function will generate an error if the file is not already open in READ mode.

Operators

add
& (Concatenation)
Joins two strings together
"Summer" & " " & "Pudding"
evaluates to "Summer Pudding"
Can also concatenate CHAR with STRING
add_circle
AND
Performs a logical AND on two Boolean values
TRUE AND FALSE
evaluates to FALSE
add_circle_outline
OR
Performs a logical OR on two Boolean values
TRUE OR FALSE
evaluates to TRUE
block
NOT
Performs a logical NOT on a Boolean value
NOT TRUE
evaluates to FALSE
percent
MOD
Finds the remainder when one number is divided by another
10 MOD 3
evaluates to 1
divide
DIV
Finds the quotient when one number is divided by another
10 DIV 3
evaluates to 3

Comparison Operators

drag_handle
= (Equal To)
Returns TRUE if two values are equal
Count = 4
evaluates to TRUE when Count contains 4
arrow_upward
> (Greater Than)
Returns TRUE if left value is greater than right value
5 > 3
evaluates to TRUE
arrow_downward
< (Less Than)
Returns TRUE if left value is less than right value
2 < 7
evaluates to TRUE
arrow_upward
>= (Greater Than or Equal)
Returns TRUE if left value is greater than or equal to right value
5 >= 5
evaluates to TRUE
arrow_downward
<= (Less Than or Equal)
Returns TRUE if left value is less than or equal to right value
3 <= 5
evaluates to TRUE
not_equal
<> (Not Equal To)
Returns TRUE if two values are not equal
"Program" <> "program"
evaluates to TRUE (case sensitive)
